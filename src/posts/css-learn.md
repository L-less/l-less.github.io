## 梗概
### 背景
`background-image:`\
`background-color:`
### 字体大小
`font-size:`\
以px为单位
## 背景
```css
<style>
body 
{
	background-image:url('paper.gif');
	background-color:#cccccc;
}
</style>
```
## 字体大小
```css
<style>
h1 {font-size:40px;}
h2 {font-size:30px;}
p {font-size:14px;}
</style>
```
